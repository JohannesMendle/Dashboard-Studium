Dies ist ein simples Dashboard, das im Zuge einer Übung zum Lernen objektorientierten Programmierens mit Python sowie UML-Diagrammen erstellt wurde. Erstellt, um einen Überblick über ein Studium zu ermöglichen, zieht es Daten aus einer .csv-Datei und stellt Status der Kurse, Notendurchschnitt sowie Fortschritt dar.

Das Skript darf frei verwendet und angepasst werden.


This is a simple dashboard that has been created as an exercise to learn object oriented programming with python as well as uml diagrams. Designed as an overview tool for your studies it uses data from a .csv file to show status, grade average as well as progress for your courses.

Feel free to use and change at will.

The script uses German language.
